# MyMusicLedger

**MyMusicLedger** is a macOS app that turns your Apple Music `Library.xml` export into a searchable, browsable catalog — artists, albums, songs, playlists, genres, and more.

Free for personal use.

---

## Download

Visit the [Releases](https://github.com/MattOnChain/MyMusicLedger-releases/releases) page to download the latest version.

| Build | Mac |
|---|---|
| **MyMusicLedger-x.x.x.dmg** | Apple Silicon (M1 / M2 / M3 / M4) |
| **MyMusicLedger-x.x.x-Intel.dmg** | Intel |

## Requirements

- macOS 13 Ventura or later
- An Apple Music `Library.xml` export file
